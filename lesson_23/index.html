Операторы JAVASCRIPT

Не верны варианты овтетов:
console.log('35' + -'22');
Так как оператор ' + ' не переведет тип данных и просто добавит одну строку к другой.
console.log('558' > 22++);
Так как инкременты работают только с переменными.
let usersCounter = 0;
let newUsers = usersCounter++;
console.log(newUsers);
Так как постфиксный инкремент "выдает" старое зачение переменной.
console.log((!false && 11) || (18 && !''));
Оператор && получает в первых скобках два значения "true" возвращается 11(так как оно последнее), во вторых скобках так же оба значения "true" и нам возвращается "true", далее оператор || сравнивает оба значения 11 и "" выбивает первое истинное - тоесть 11.
let name = 0;
console.log(name ?? 'Без имени');
Вернет нам переменную name, тоесть 0, так как ёё значение отличное от null или undefined.

IF ELSE в JAVASCRIPT

Ответы:
if (1 === '1') {
console.log('true');
} else {
console.log('false');
}
Вернет нам 'false' потому что строгое сравнение '===' не переводит тип данных
if (5 == '5') {
console.log('true');
} else {
console.log('false');
}
Вернет нам true, так как нестрогое сравнение '==' конвертирует оба значения в один тип данных - числовой
let message = (92 > '11' && 54 < 100) ? 'true' : 'false' ; console.log(message);
Вернет нам 'true', потому что для сравнения операторы '<' '>' приводят значения к одному типу данных и общее выражение в скобках становистя правдиво.
if (0) {
console.log('true');
} else if (' ') {
console.log('false');
}
Вернет 'false', потому что код помещенный в if выполнится только при получении значения 'true' в условиях, а нашем случае там находится 0 = false, второе условие выполняется, так как строка с пробелом имеет значение 'true'
